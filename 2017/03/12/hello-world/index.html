实验一 在Github上用Markdown创建技术博客

  • 一、实验目的
  • 二、实验任务
  • 三、实验步骤与结果
  • 四、实验小结

一、实验目的

  1. 熟悉Github网站
  2. 掌握Markdown语法

    二、实验任务

  3. 注册Github账号
  4. 创建Gihub Pages repository
  5. 发布第一篇博客
  6. 学习Markdown
  7. 利用Hexo生成博客并部署到Github

    三、实验步骤与结果

一、本地配置Hexo

  1. 安装git
  2. 安装node
  3. 设置npm镜像为淘宝
    npm install -g cnpm –registry=https://registry.npm.taobao.org
  4. 安装hexo
    npm install -g hexo-cli
  5. 初始化站点
    hexo init myhexo
    cd myhexo
    npm install
  6. 浏览本地站点
    hexo server

    二、在Github上部署hexo

  7. 在Github上新建repository，名称是guqinqing.github.io

  8. 修改配置文件_config.yml的deploy部分，注意空格

  9. 配置site部分

  10. 安装 hexo-deployer-git
    npm install hexo-deployer-git –save

  11. 配置git全局邮箱与名字
    git config –global user.email “936028856@gmail.com”
    git config –global user.name “guqinqing”
  12. 生成博客
    hexo generate
  13. 部署博客
    hexo deploy

    三、添加新文章

  14. 生成新文章
    hexo new newblog
  15. 用编辑器修改文章：myhexo\source_posts\newblog.md
  16. 生成并发布到Github
    hexo d -g

四、实验小结

通过本次实验让我熟悉Github网站，学习了Markdown语法，发布了第一篇博客。开拓了我的眼界，提升了我的自主学习能力，同时也让我意识到自身的不足，还需更深入的学习相关知识。
